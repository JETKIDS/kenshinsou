# 古式健身操アプリ Design

**Goal:** Gemini などで編集した短尺エクササイズ動画をプロジェクト内に置き、毎回ランダムに5動作を選んで、スマホで見ながら約5分運動できるWEBアプリを作る。

## Scope

- React / Vite のWEBアプリとして開発する。
- 最初はスマホ縦画面を優先する。
- 動画素材は `public/videos/` にまとめて置く。
- 初期状態では仮の動画ファイル名をデータとして持ち、実ファイルは後から差し替えられるようにする。
- 20種類前後の動作データから、ランダムに5動作を選ぶ。
- 選ばれた5動作を一覧で確認し、順番に動画を再生できるようにする。

## Architecture

- `src/data/exercises.ts` に動作データを置く。
- `src/lib/selectWorkout.ts` にランダム選択ロジックを置く。
- `src/App.tsx` は画面状態を持ち、メニュー生成と再生対象の切り替えを扱う。
- `public/videos/` は編集済み動画ファイルの配置場所にする。

## Initial UX

- 画面上部にアプリ名と短い状態表示を置く。
- メイン領域に現在選択中の動画プレイヤーを置く。
- 下部に「今日の5動作」を生成するボタンと、選ばれた5動作のリストを置く。
- リスト項目を押すと、その動作の動画を再生対象にする。

## Testing

- ランダム選択ロジックは Vitest でテストする。
- 5件選ばれること、重複しないこと、元データ数が少ない場合は可能な件数だけ返すことを確認する。
- UI は初期実装ではビルドと目視確認を中心に検証する。

## Out of Scope for First Version

- 動画の自動連続再生
- PWA 化
- ユーザー別の練習履歴
- 外部ストレージ連携
- 動画編集機能
